# API描述

**接口地址：** http://cbnxzs.com/api

**请求方式：** GET

**返回格式：** 图片

**参数说明：**

|名称|必填|类型|说明|
| :-: | :-: | :-: | --- |
|id|否|string|取值为1～6，返回下标固定的图片，不使用该参数则返回随机图片。默认不使用该参数。|

**请求示例：**

|示例|说明|
|:-:|:-:|
|http://cbnxzs.com/api|返回随机的一张图片|
|http://cbnxzs.com/api?id=1|返回下标为1的一张图片|
